
Banking App
===========

Spring Boot + REST + Validation + Spring Data JPA

REST API
	
	Account Service [account-service]
	GET 	/accounts
	GET		/accounts/{id}
	POST	/accounts
	PUT		/accounts/{id}
	DELETE	/accounts/{id}
	
	Transaction Service [transaction-service]
	GET 	/transactions
	GET		/transactions/{id}
	POST	/transactions
	PUT		/transactions/{id}
	DELETE	/transactions/{id}
	
	Fund Transfer Service	[fund-transfer-service]
	POST 	/fundTransfers
